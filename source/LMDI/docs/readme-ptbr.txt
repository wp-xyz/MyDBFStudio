Suite LMDI

A Suite LMDI ("Lazarus MDI" Interface Simulation ou Simulação de Inteface MDI para Lazarus) é composta por componentes para fazer uma simulação de aplicação MDI.
Ela é totalmente escrita baseando-se em componentes já contido na VCL/LCL (TPanel, TImage, etc).
A Suite LMDI contém os seguintes componentes:

TButtonsBar = Uma barra de botões para minimizar, restaurar e fechar as janelas filhas (também pode ser usada para outros propósitos)
TFormPanel = Uma espécie de "janela-filha", que será utilizada como esqueleto para o componente TChildDoc (veja MultiDoc)
TTitleBar = Uma barra de títulos, descendente de TButtonsBar, que será utilizada nas janelas filhas e arrastará essas janelas no container (TMultiDoc)
